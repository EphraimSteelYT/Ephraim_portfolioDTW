# Favicon Files for Ephraim W. Prayoga Portfolio

This folder contains the favicon files for the portfolio website.

## Files:
- `favicon.ico` - Main favicon file (traditional ICO format)
- `favicon.svg` - Modern SVG favicon with "EP" initials
- `favicon-16x16.png` - 16x16 PNG favicon
- `favicon-32x32.png` - 32x32 PNG favicon
- `apple-touch-icon.png` - Apple touch icon for iOS devices (180x180)

## Usage:
These files are linked in the HTML head sections of all pages:
- Index.html (main portfolio)
- lego.html
- militarytechnology.html
- youtube.html
- politics.html

The favicon will appear in browser tabs, bookmarks, and shortcuts.
